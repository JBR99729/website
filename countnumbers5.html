Counting Numbers 1-5

These worksheets help nursery children practise counting pictures and matching them with numbers from 1 to 5.

How to Complete These Worksheets

Look at the pictures carefully.

Count them one by one.

Circle, match, write, or colour the correct number.

Worksheet 1 – Count and Circle

Count the pictures and circle the correct number.

Example: If you see 3 stars, circle 3.
Download Worksheet

Worksheet 2 – Count and Match

Match each group of pictures to the correct number.

Example: 4 apples → number 4.
Download Worksheet

Worksheet 3 – Count and Write

Count the pictures and write the correct number.

Example: 2 balls → write 2.
Download Worksheet

Worksheet 4 – Count and Colour

Count the objects and colour the correct number.

Example: Colour 5 flowers.
